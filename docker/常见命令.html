    # 常见命令

    1. 容器列表
docker ps 或者 docker ps -a
2. 删除容器
docker stop containerId (先停止)
docker rm containerId 
3. 镜像列表
docker images
4. 删除镜像
docker rmi imageId
5. 构建
docker build -t node/test-server . 
-t 后面跟名称
. 表示当前目录(重要)
6. 启动
docker run -d --name test-server -p 9002:9002 node/test-server
-d 后台运行容器
--name 名称
-p 端口 dockerfile暴露的端口保持一致
7. 查看是否成功
docker ps
8. 查看日志
docker logs containerId
9. 重命名镜像名称
docker tag nexus.in.wezhuiyi.com:5000/yibot/base:2.0.5 yibot/base:2.0.5
用于 from yibot/base:2.0.5 失败的情况
